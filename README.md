# Personal-site
Responsive personal portfolio built using Gatsby, TypeScript, and TailwindCSS

Check it out here: [daniel-park.netlify.app](https://daniel-park.netlify.app)

# Want to run it yourself?
```
yarn
yarn run develop
```
